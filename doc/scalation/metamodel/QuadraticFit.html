QuadraticFit

class QuadraticFit extends AnyRef

This class uses multiple regression to fit a quadratic surface to the function f. This is useful when computing f is costly, for example in simulation optimization. The fit is over a multi-dimensional grid and can be used for interpolation and limited extrapolation.
